This is a file containing version info as expanded by subversion
It is read by config.py to obtain the version number for pyvolution.


------------------------------------- 	
To set subversion keyword expansion:

svn propset svn:keywords "LastChangedDate LastChangedRevision LastChangedBy HeadURL Id" version.txt

Subversion keywords:

$LastChangedDate$
$LastChangedRevision$
$LastChangedBy$
$HeadURL$
$Id$
